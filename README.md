# Calculadora ACT - solução web responsiva

## Arquivos
- `index.html`
- `styles.css`
- `script.js`

## Como usar
1. Abra o arquivo `index.html` em qualquer navegador moderno.
2. A tela já abre com o exemplo ACT preenchido automaticamente.
3. Edite os valores da calculadora para testar outros cenários.

## Recursos incluídos
- menu responsivo
- animações suaves
- gráficos dinâmicos com Chart.js
- botões de cenário rápido
- resumo executivo com copiar e colar
- layout otimizado para desktop e mobile

## Premissas do cálculo
- Baseado no exemplo da apresentação da ACT.
- Tributos considerados: IR 15%, adicional de IR 10% e CSLL 9%.
- Simulação com foco em empresas de Lucro Real.
